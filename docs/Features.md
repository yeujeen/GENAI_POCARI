# 상세 기능 명세

## F-01: 사용자 데이터 입력

* **기능:** 두 사용자가 각자의 음악 취향 데이터를 서비스에 입력한다.
* **MVP 요구사항:**
    * User A와 User B가 각각의 텍스트 입력창에 자신의 대표 취향 (예: 좋아하는 아티스트 5명, 장르 3개)을 직접 입력한다.
    * 'FUSE!' 버튼을 누르면 입력된 데이터가 서버로 전송된다.
* **추후 확장:**
    * Spotify 등 외부 음악 서비스 API와 연동하여 청취 데이터를 자동으로 가져온다.

## F-02: 취향 분석 및 유사도 계산

* **기능:** AI가 입력된 두 사용자의 데이터(장르, 아티스트, 태그)를 분석하여 취향 유사도를 %로 계산한다.
* **MVP 요구사항:**
    * 서버는 전송된 텍스트 데이터를 AI 모델(Gemini)에 전달한다.
    * AI는 두 데이터의 교집합(공통 아티스트, 장르)을 분석하여 '유사도' 수치(예: 45%)를 계산하여 반환한다.

## F-03: FUSE Mix (공동 플레이리스트) 생성

* **기능:** AI가 분석된 교집합을 기반으로 두 사용자가 함께 들을 최적의 플레이리스트(10~15곡)를 자동 생성한다.
* **MVP 요구사항:**
    * AI는 유사도와 함께, 교집합 영역의 곡과 상호 추천 곡으로 구성된 'FUSE Mix' (10곡)를 생성한다.
    * AI가 '팀플 집중 믹스' 같이 생성된 믹스에 어울리는 **독창적인 제목**과 **한 줄 설명**을 생성하여 함께 반환한다.

## F-04: 결과 시각화 및 표시

* **기능:** 분석된 유사도와 생성된 플레이리스트를 사용자에게 시각적으로 보여준다.
* **MVP 요구사항:**
    * 결과 페이지에 계산된 **유사도(%)**가 명확하게 표시된다.
    * 두 사용자의 취향 교집합을 표현하는 **벤 다이어그램** 이미지가 표시된다. (초기에는 정적 이미지, 추후 동적 그래프)
    * 생성된 'FUSE Mix'의 AI 추천 제목, 한 줄 설명, 10개의 곡 목록(아티스트, 곡명)이 표시된다.

## F-05: 결과 공유

* **기능:** 생성된 FUSE Mix 결과 페이지를 다른 사람에게 공유한다.
* **MVP 요구사항:**
    * 결과 페이지에 '공유하기' 버튼이 존재한다.
    * 클릭 시, 현재 결과 페이지를 볼 수 있는 **고유 URL**이 클립보드에 복사된다.
